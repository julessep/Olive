# Olive

Olive is a recipe finder web app that allows users to search for recipes by diet type, ingredient or key word. This was my frontend capstone project for NSS.

# Software Requirements 

- [Node.js](https://nodejs.org/en/)
- [npm](https://www.npmjs.com/)

# Technologies Used

- [AngularJS](https://angularjs.org/)
- [Firebase](https://firebase.google.com/)
- [Food2Fork API](https://food2fork.com/about/api)
